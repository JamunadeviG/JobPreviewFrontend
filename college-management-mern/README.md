# College Management MERN Stack

## Prerequisites
- Node.js 18+
- MongoDB 6+ running locally

## Quick Start

### 1. Backend
```bash
cd backend
cp .env.example .env  # customize if needed
npm install
npm run dev
```

### 2. Frontend
```bash
cd ../frontend
npm install
npm run dev
```

Open http://localhost:5173

> The React app will fetch from `http://localhost:8080/api`.
